## Navigation

| Command | Description | Syntax |
| ------- | ----------- | ------ |
| `cd` | Change Directory | `cd [directory path]` |
| `ls` | List contents of directory	 | `ls [options] [directory path]` |
| `pwd` | Print working directory | `pwd` |
